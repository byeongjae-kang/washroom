## Getting Started

### Setup

* Install Expo CLI 
```sh
npm install --global expo-cli
```

* Clone Repository
```sh
$ git clone git@github.com:byeongjae-kang/washroom.git
$ cd washroom
```

* Install Dependencies
```sh
npm install
```

* Run Expo
```sh
npm start
```
